project name: pet-care-in-winter
Purpose:
"The primary purpose of WarmPaws is to provide a comprehensive and user-friendly platform for pet owners to ensure their pets' well-being during the challenging winter months. This project aims to simplify the process of finding specialized winter pet care services such as grooming, clothing, and health consultations. By integrating secure authentication, dynamic service details, and personalized user profiles, WarmPaws creates a reliable ecosystem where pet lovers can access expert advice and essential services, ensuring their furry companions stay warm, safe, and happy throughout the season."

Key Features:
Winter-Themed Responsive Design: A modern, minimalist UI specifically designed for winter pet care, ensuring a seamless experience across Mobile, Tablet, and Desktop devices.

Service Management & Dynamic Routing: Users can explore a variety of pet care services. Each service has a dedicated "View Details" page that is protected, ensuring only authenticated users can access deep information.

Robust Authentication System: Integrated Firebase Authentication providing secure Email/Password registration and login, along with a one-click Google Social Login feature.

Advanced User Profile Management: A dedicated "My Profile" page where users can view their information and dynamically update their Name and Profile Picture using a functional update form.

Smart Password Recovery: A functional "Forgot Password" system that pre-fills the user's email from the login page and redirects them to Gmail to complete the reset process.

Interactive UI with Animations: Enhanced user experience using AOS (Animate On Scroll) for smooth entry effects and Swiper.js for an engaging hero slider on the homepage.

Real-time Feedback: Integrated React-Toastify to provide instant success or error notifications for actions like login, registration, and profile updates.

Security & Best Practices: All sensitive Firebase configuration keys are secured using Environment Variables (.env) to prevent unauthorized access.
